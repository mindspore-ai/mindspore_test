mindspore.ops.swapdims
=======================

.. py:function:: mindspore.ops.swapdims(input, dim0, dim1)

    交换tensor的两个维度。
    该函数和 :func:`mindspore.ops.swapaxes` 功能一致。

    参数：
        - **input** (Tensor) - 输入tensor。
        - **dim0** (int) - 第一个维度。
        - **dim1** (int) - 第二个维度。

    返回：
        Tensor
