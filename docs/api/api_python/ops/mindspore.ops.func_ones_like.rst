mindspore.ops.ones_like
=======================

.. py:function:: mindspore.ops.ones_like(input, *, dtype=None)

    返回一个shape与输入相同，值为1的tensor。

    参数：
        - **input** (Tensor) - 输入tensor。

    关键字参数：
        - **dtype** (:class:`mindspore.dtype`, 可选) - 指定数据类型。默认 ``None`` 表示与 `input` 数据类型相同。

    返回：
        Tensor
