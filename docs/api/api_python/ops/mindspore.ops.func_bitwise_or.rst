mindspore.ops.bitwise_or
========================

.. py:function:: mindspore.ops.bitwise_or(input, other)

    逐元素对两个输入tensor做或运算。
    如果 `input` 和 `other` 数据类型不同，遵循隐式类型转换规则、类型提升规则。

    参数：
        - **input** (Tensor) - 第一个输入tensor。
        - **other** (Tensor) - 第二个输入tensor。

    返回：
        Tensor
