mindspore.mint.swapaxes
==========================

.. py:function:: mindspore.mint.swapaxes(input, axis0, axis1)

    :func:`mindspore.mint.transpose` 的别名。其中参数 `input` 对应参考接口的 `input` ，参数 `axis0` 和 `axis1` 分别对应参考接口的 `dim0` 和 `dim1` 。
    
    更多参考详见 :func:`mindspore.mint.transpose`。

    .. warning::
        这是一个实验性API，后续可能修改或删除。
