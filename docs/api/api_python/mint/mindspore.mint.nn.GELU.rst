mindspore.mint.nn.GELU
====================================

.. py:class:: mindspore.mint.nn.GELU(approximate='none')

    高斯误差线性单元激活函数（Gaussian Error Linear Unit）。

    GELU的详细描述可以参考论文： `Gaussian Error Linear Units (GELUs) <https://arxiv.org/abs/1606.08415>`_ ，
    或论文： `BERT: Pre-training of Deep Bidirectional Transformers for Language Understanding <https://arxiv.org/abs/1810.04805>`_ 。

    更多参考详见 :func:`mindspore.mint.nn.functional.gelu`。
