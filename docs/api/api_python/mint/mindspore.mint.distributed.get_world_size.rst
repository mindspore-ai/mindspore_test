mindspore.mint.distributed.get_world_size
============================================

.. py:function:: mindspore.mint.distributed.get_world_size(group=None)

    获取指定通信组实例的rank_size。

    .. note::
        `get_world_size` 方法应该在 `mindspore.mint.distributed.init_process_group` 方法之后使用。

    参数：
        - **group** (str，可选) - 通信组名称，通常由 `mindspore.mint.distributed.new_group` 方法创建，如果为 ``None`` ， Ascend平台表示为 ``"hccl_world_group"`` 。 默认值： ``None`` 。

    返回：
        指定通信组实例的rank_size，数据类型为int。

    异常：
        - **TypeError** - 在参数 `group` 不是字符串时抛出。
