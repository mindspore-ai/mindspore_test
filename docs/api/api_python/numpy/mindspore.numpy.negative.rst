mindspore.numpy.negative
========================

.. py:function:: mindspore.numpy.negative(a, dtype=None)

    数值符号取反，逐元素操作
    
    .. note::
        不支持NumPy参数 `out` 、 `where` 、 `casting` 、 `order` 、 `subok` 、 `signature` 、 `extobj` 。

    参数：
        - **a** (Tensor) - 输入Tensor。
        - **dtype** (mindspore.dtype, 可选) - 默认值: `None` 。 覆盖输出Tensor的dtype。

    返回：
        Tensor。
